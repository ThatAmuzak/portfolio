# Rishav Banerjee — Portfolio

Single-page portfolio built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com). YAML-driven content — edit `src/data/`, deploy, done.

**[rishav-banerjee.com](https://rishav-banerjee.com)**

## Quick Start

```bash
npm install
npm run dev
```

## Structure

```
src/
├── components/   # Astro components per section
├── data/         # YAML content files
├── layouts/      # BaseLayout with SEO, OG, JSON-LD
├── lib/          # Data loader, types, helpers
├── pages/        # index.astro
└── styles/       # global.css (Tailwind theme, animations)
```
